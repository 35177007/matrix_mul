## 2. 数据说明
1. 从文件读入两个矩阵 A 和 B，数据类型仅为整型。
2. A 和 B 应满足矩阵乘法的条件；否则程序会捕获异常并输出异常信息。
3. 矩阵每一行以**回车**结尾，**最后一行**也应如此。
### 2.1 矩阵 A 示例
  ```
  // a.txt 
  2 3 13 34
  4 23 44 111
  1 2 3 4
    
  ```
### 2.2 矩阵 B 示例
  ```
  // b.txt
  1 2 3 
  5 6 7 
  9 10 11
  12 13 14
  
  ```
